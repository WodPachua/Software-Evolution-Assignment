# Contributing

Thanks for your interest in improving Task Manager.

## How to contribute
1. Fork the repository and create a feature branch (`feature-xyz`).
2. Make small, focused commits with clear messages.
3. Run the program and make sure it still works before pushing.
4. Open a Pull Request describing what you changed and why.

## Commit message style
- Use the imperative mood: "Add search command", not "added search".
- Keep the first line under ~70 characters.
- Explain the "why" in the body if it is not obvious.

## Reporting bugs
Open a GitHub Issue with steps to reproduce, what you expected, and what
actually happened.
